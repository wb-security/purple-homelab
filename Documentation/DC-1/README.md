DC-1 is the primary domain controller for the **purple.local** Active Directory forest.  
It manages authentication, DNS, organizational structure, user accounts, and Group Policy for the homelab.

This documentation covers:
- Deployment of Windows Server 2019  
- Active Directory Domain Services installation  
- DNS configuration  
- Domain promotion  
- Administrative user creation  
- Organizational Unit (OU) structure  
- Validation and testing  
- Troubleshooting issues encountered during setup  

DC-1 serves as the backbone of the lab environment and is the first VM configured in the purple-homelab project.
